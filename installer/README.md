# Upper Room Assembly - Church Management System Installer

## How to Build

1. Download and install Inno Setup from https://jrsoftware.org/isdl.php
2. Open `URA_CMS_Installer.iss` with Inno Setup Compiler
3. Click Build > Compile
4. The installer EXE will be created in the `Output` folder!

## Features for Users

- **New Users**: Installs a standalone web server + your app
- **Existing Users**: Detects existing installations and updates
- Auto-launches app after installation
- Creates desktop shortcuts and start menu entries
